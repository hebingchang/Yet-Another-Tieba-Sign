# Yet Another Tieba Sign

## Requirement

- PHP 7+
- MySQL
- Apache or Nginx
- NodeJS

## Architecture

- Laravel
- Vue

## Start Development

- Install PHP composer
- Install Node and npm
- Run `composer install` and `npm install`
- Modify `.env` to configure the database
- `php artisan migrate`
- `php artisan serve` and `npm run watch`
- Start developing!
